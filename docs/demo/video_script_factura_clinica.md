# FiscoBot — Guion de Video Explicativo (60 a 90 Segundos)

Este guion está estructurado para un video comercial rápido de alta conversión, diseñado para presentarse en redes sociales, páginas de aterrizaje o correos de prospección. Está dirigido a dueños de clínicas, administradores y despachos contables.

*   Este video describe un **flujo de trabajo y validación de datos en una demo cloud controlada**.
*   **Nunca** se debe mencionar timbrado automático, validación SAT automática ni automatización total de WhatsApp mediante APIs.
